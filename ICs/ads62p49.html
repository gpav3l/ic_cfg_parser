ADS62P49 Config parser/creator

Config register value, write in HEX format. To generate new confic, change value in comboboxes.

REG00 (0x00)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
0
(SERIAL READOUT)
7
(RESET)		 Set by software, clear by hardware

REG01 (0x20)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
2
(ENABLE LOW SPEED MODE)

REG02 (0x3F)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
1
(STANDBY)
6:5
(REF. Internal or external reference selection)

REG03 (0x40)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
3:0
(POWER DOWN MODES)

REG04 (0x41)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
7
(LVDS CMOS)

REG05 (0x44)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
4:2
(CLKOUT POSN) 		LVDS - Output clock falling edge position
CMOS - Output clock rising edge position 		Keep the same duty cycle, move both edges by the same amount(i.e. write the same value to 7:5)
7:5
(CLKOUT POSN) 		LVDS - Output clock rising edge position
CMOS - Output clock falling edge position 		Keep the same duty cycle, move both edges by the same amount(i.e. write the same value to 4:2)

REG06 (0x50)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
2:1
(DATA FORMAT)
6
(ENABLE INDEPENDENT CHANNEL CONTROL)

REG07 (0x51)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
7:0
(CUSTOM PATTERN LOW)		 8 LSB bit for custom pattern used with "Test pattern" reg 0x62

REG08 (0x52)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
5:0
(CUSTOM PATTERN HIGH)		 6 MSB bit for custom pattern used with "Test pattern" reg 0x62

REG09 (0x53)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
6
(ENABLE OFFSET CORRECTION – Common/Ch A)

REG10 (0x55)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
3:0
(OFFSET CORR TIME CONSTANT – Common/Ch A)
7:4
(GAIN – Common/Ch A)

REG11 (0x57)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
6:0
(FINE GAIN ADJUST)		 FINE GAIN ADJUST – Common/Ch A Channel gain = 20*log10[1 + (FINE GAIN ADJUST/8192)]

REG12 (0x62)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
2:0
(TEST PATTERNS) 		Config for channel A only if independet mode is enabled, for both channels in other case

REG13 (0x63)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
5:0
(OFFSET PEDESTAL – Common/Ch A)

REG14 (0x66)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
6
(ENABLE OFFSET CORRECTION – CH B)

REG15 (0x68)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
3:0
(OFFSET CORR TIME CONSTANT – CH B)
7:4
(GAIN – CH B)

REG16 (0x6A)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
6:0
(FINE GAIN ADJUST)		 FINE GAIN ADJUST – CH B Channel gain = 20*log10[1 + (FINE GAIN ADJUST/8192)]

REG17 (0x75)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
2:0
(TEST PATTERNS) 		Config for channel B only if independet mode is enabled

REG18 (0x76)

REGISTER BIT RELATED BLOCK FUNCTION COMMENT
5:0
(OFFSET PEDESTAL – Common/CH B)
